## Human Resources System with Clock-in Clock-out Overtime

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/455d6fdd906645a997d7be4aefc40349/ghb20250918) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Attendance Information Management
6. Clock-in Record Management
7. Clock-out Record Management
8. Overtime Record Management
9. Employee Salary Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Department Information Management
4. Employee Information Management
5. Attendance Information Management
6. Clock-in Record Management
7. Clock-out Record Management
8. Overtime Record Management
9. Employee Salary Management
